# Aplicacion_de_Arbol_Binario_de_Busqueda_en_Java-Gestion_de_Tarjetas_DC
Este proyecto consiste en una aplicación de escritorio desarrollada en Java Swing, que implementa desde cero un Árbol Binario de Búsqueda (ABB) para gestionar una colección de tarjetas de DC Comics.  La aplicación permite insertar, buscar, eliminar, recorrer y visualizar los datos mediante una interfaz gráfica interactiva.
